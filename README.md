# 🚀 Vnet – Smart Networking Platform

Vnet is a full-stack social networking web application that enables users to connect, communicate, and share content in real-time. It provides a seamless user experience with features like authentication, chat system, posts, and notifications.

---

## 🌟 Features

* 🔐 User Authentication (Login / Signup / Forgot Password)
* 🧑 User Profile Management
* 📝 Create, Edit, and Delete Posts
* 💬 Real-time Chat System
* 🔔 Notifications System
* 📸 Image Upload Support
* 🧭 Responsive UI with smooth navigation

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Utkarsh1021/Vnet.git
cd Vnet
```

### 2️⃣ Install dependencies

#### Frontend

```bash
cd frontend/Vnet
npm install
npm run dev
```

#### Backend

```bash
cd ../backend
npm install
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file in backend folder and add:

```
MONGO_URI=******************  
JWT_SECRET=****************  
```
